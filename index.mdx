## Get started in 3 steps

<Steps>
  <Step title="Install PyMuPDF">
    Install the library using pip. No external dependencies required.

    ```bash
    pip install pymupdf
    ```
  </Step>
  <Step title="Open a document">
    Use `pymupdf.open()` to load any supported file type.

    ```python
    import pymupdf

    doc = pymupdf.open("document.pdf")
    print(f"Document has {doc.page_count} pages")
    ```
  </Step>
  <Step title="Extract text or render pages">
    Iterate over pages to extract text or render them as images.

    ```python
    for page in doc:
        text = page.get_text()
        print(text)

    # Or render a page to an image
    page = doc[0]
    pix = page.get_pixmap(dpi=150)
    pix.save("page-1.png")
    ```
  </Step>
</Steps>
